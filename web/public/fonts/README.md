# Fonts

## Alagard

`alagard.ttf` is the Alagard pixel font by Hewett Tsoi, a bitmap gothic font designed for game projects. We use it for all mechanical readouts in the UI (chaos numeral, dice numbers, oracle tags, kicker labels) — it's the visual voice of the engine, while `Cormorant Garamond` carries the narrative voice and `IM Fell English SC` carries scene heads.

License: Creative Commons Attribution (CC-BY). The author's note: _"Give credit if used. That's it."_

Source: https://www.dafont.com/alagard.font

If `alagard.ttf` is ever missing, the CSS falls back to `VT323` (a similar pixel font from Google Fonts) so the engine voice is still pixelated, just less distinctively gothic.
